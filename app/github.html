1
Inicializar el Repositorio

Crea un repositorio Git local en la carpeta de tu proyecto para empezar a rastrear los archivos.

git init
2
Preparar los Archivos

Añade los archivos modificados o nuevos al área de preparación (Staging Area) antes de guardarlos.

git add .
3
Confirmar Cambios

Guarda una "foto" o versión actual de tus archivos en el historial local con un mensaje descriptivo.

git commit -m "Mi primer commit"
4
Crear la Rama Principal

Asegúrate de que el nombre de tu rama principal sea main (el estándar de GitHub).

git branch -M main
5
Vincular a GitHub

Conecta tu repositorio local con el repositorio remoto vacío que creaste previamente en GitHub.

git remote add origin URL_DE_GITHUB
6
Subir el Código

Empuja tus archivos locales hacia la nube de GitHub por primera vez usando el parámetro -u.

git push -u origin main
