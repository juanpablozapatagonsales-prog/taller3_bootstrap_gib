# ☕ Coffee Code - Taller de Maquetación Web con Bootstrap 5

¡Bienvenido a **TALLER #3**! Este es un proyecto de maquetación web enfocado en la creación de interfaces limpias, adaptables y accesibles, desarrollado como parte de las evidencias de formación técnica en desarrollo de software.

La plataforma simula un entorno web administrativo y de presentación de servicios, aplicando componentes avanzados del framework **Bootstrap 5**.

---

## 🚀 Características Principales

- **Diseño 100% Responsivo:** Adaptado completamente para dispositivos móviles, tablets y pantallas de escritorio mediante el sistema de rejilla (*Grid System*) de Bootstrap.
- **Navegación Interactiva:** Menú superior (`Navbar`) fijo con menús desplegables estructurados para *Módulos* (Usuarios, Productos, Inscripciones) y *Consultas* (Historial, Reportes).
- **Sección de Datos Dinámica:** Maquetación de tablas de datos (`app/tables.html`) personalizadas con estilos responsivos y combinación de colores accesibles.
- **Formularios con Validación:** Interfaz limpia para el inicio de sesión y registro de usuarios con iconografía descriptiva (`Bootstrap Icons`).
- **Carrusel Adaptativo:** Slider dinámico para la visualización de imágenes principales en la sección de inicio.

---

## 🛠️ Tecnologías Utilizadas

Las herramientas y tecnologías principales implementadas en este taller son:

* **HTML5:** Estructuración semántica del documento.
* **CSS3:** Personalización de estilos y maquetación complementaria.
* **Bootstrap v5.3.3:** Framework ágil para el diseño adaptable y componentes interactivos.
* **Bootstrap Icons v1.13.1:** Librería tipográfica de iconos del sistema.
* **JavaScript (Bundle de Bootstrap):** Manejo de animaciones, colapsables y menús desplegables.
